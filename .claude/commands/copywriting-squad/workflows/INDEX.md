## Fluxo de Uso Recomendado

**Para lançamento completo:**
```
1. Oferta Irresistível (criar oferta)
2. VSL Matador (criar vídeo de venda)
3. Máquina de Leads (capturar leads)
4. Webinar que Vende (converter leads)
5. Email Diário (monetizar lista)
```

**Para tráfego pago rápido:**
```
1. Oferta Irresistível (definir oferta)
2. Ads que Escalam (criar criativos)
3. Máquina de Leads (capturar leads)
```

**Para monetizar lista existente:**
```
1. Reativação de Lista (limpar e engajar)
2. Email Diário (vender consistentemente)
```

## Como Usar um Workflow

1. **Escolher workflow** baseado no objetivo
2. **Ler PRE-CHECKLIST** e garantir que tem os inputs
3. **Seguir as FASES** na ordem
4. **Usar os CLONES** indicados para cada fase
5. **Validar com POS-CHECKLIST** antes de publicar
6. **Entregar OUTPUT FINAL** completo

---

## Estrutura de Cada Workflow (Framework 9 Passos)

```
PASSO 1: INPUT
O que precisa ter antes de começar

PASSO 2: INPUT VALUE
Formato específico dos inputs

PASSO 3: PRE-CHECKLIST
Quality gate de entrada

PASSO 4: PERSONA/EXECUTOR
Qual clone executa cada parte

PASSO 5: PROMPT DE EXECUÇÃO
Instruções detalhadas por fase

PASSO 6: TOOLS/SCRIPTS
Ferramentas e dependências

PASSO 7: OUTPUT
O que sai (formato exato)

PASSO 8: POS-CHECKLIST
Quality gate de saída

PASSO 9: OUTPUT FINAL
Resultado "bem fudido"
```
