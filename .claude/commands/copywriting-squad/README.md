# Copywriting Masters Squad

## Overview

A **Copywriting Masters Squad** e um expansion pack que organiza **15 clones de copywriters lendarios** em um sistema hierarquico de 3 niveis para criar copy de alta conversao.

## Filosofia

Este framework e baseado em um principio simples:

- Alguns copywriters sao melhores para **PENSAR** a estrategia (Tier 1)
- Outros para **EXECUTAR** a escrita (Tier 2)
- E outros para **OTIMIZAR** o resultado (Tier 3)

**Principio Central:** Nunca comece a escrever copy sem passar por pelo menos 1 estrategista.

## Workflows Disponiveis

### 1. Lancamento Completo (5-7 dias)
Workflow completo para lancamento de produto, do diagnostico a otimizacao.

### 2. Trafego Pago Rapido (2-3 dias)
Ads + Landing Page + Sequencia de emails basica.

### 3. High-Ticket Sales
Estrategia completa para vendas acima de R$5.000.

### 4. Conteudo Organico Diario
Posts, threads, newsletters para construcao de audiencia.

### 5. Email Marketing Continuo
Escolha entre estilo Settle (diario) ou Chaperon (automacao).

### 6. Otimizacao de Funil Existente
Audit e melhoria de copy que nao esta convertendo.
